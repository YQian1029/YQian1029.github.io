二、C++ 变量

变量是用于存储数据值的容器。若要声明变量，请指定类型并为其赋值。注：

  • 系统会为变量开辟空间，空间大小为该变量类型的大小。
  • 变量名代表空间的内容，对变量名的操作等价于对空间的操作。

1.变量类型

  • int- 存储整数（整数），不带小数，例如 ：int a= 5;
  • double- 存储带有小数的浮点数，例如 : double a=19.99;
  • char- 存储单个字符，例如 ：char a= ‘D’;
  • string- 存储文本，例如：string a = “Hello World”;
  • bool- 存储具有两种状态的值： 对或错。bool a =true;

2.识别符

所有C++变量都需要用唯一名称进行标识，这个唯一名称我们定义为标识符。

命名变量的一般规则是：

  • 名称可以包含字母、数字和下划线
  • 名称必须以字母或下划线 （_） 开头
  • 名称区分大小写（并且是不同的变量）myVar myvar
  • 名称不能包含空格或特殊字符，如 ！、#、% 等。
  • 保留字（如 C++ 关键字，如 ）不能用作名称int

3.常量

使变量保持不变，表示不可更改或只读。

  1. define 宏常量： #define 常量名 常量值

  2. const修饰的变量 const 数据类型 常量名 = 常量值
